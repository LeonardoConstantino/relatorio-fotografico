# Task 17: Edição Avançada de Imagem (Crop, Rotate, Meta)

## Objetivo
Transformar o modal de edição de imagem em uma ferramenta completa de tratamento de evidências.

## Detalhamento
1.  **Ferramentas de Transformação:**
    *   Botão **Girar 90°**: Rotacionar o canvas internamente.
    *   Botão **Cortar (Crop)**: Interface simples para redimensionar a área de foco da foto.
2.  **Painel de Metadados:**
    *   Adicionar campos no modal para editar:
        *   Data/Hora da captura (manual ou via EXIF se disponível).
        *   Coordenadas GPS ou Localização Técnica.
3.  **Estamparia (Watermark):**
    *   Opção de "Queimar" os metadados no canto inferior da imagem (Data/Hora e Trace ID) para auditoria.

## Critérios de Aceite
- [ ] Foto tirada de lado pode ser corrigida no modal.
- [ ] Metadados aparecem de forma legível na imagem ou logo abaixo dela no PDF.
