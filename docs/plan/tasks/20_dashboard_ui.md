# Task 20: Interface de Dashboard (Listagem e Busca)

## Objetivo
Criar a tela inicial do aplicativo onde o usuário gerencia seu arquivo de relatórios.

## Detalhamento
1.  **Tela de Dashboard:**
    *   Grade de cards ou lista técnica com todos os relatórios salvos.
    *   Barra de busca por nome da empresa ou número.
2.  **Fluxo de Entrada:**
    *   Ao abrir o app, se não houver um "relatório ativo", exibir o Dashboard.
3.  **Estética:**
    *   Seguir o visual "Studio" Dark Mode.

## Critérios de Aceite
- [ ] Usuário clica em um relatório da lista e ele abre no Editor/Preview instantaneamente.
- [ ] Botão "Novo Relatório" no Dashboard cria um novo registro em branco.
