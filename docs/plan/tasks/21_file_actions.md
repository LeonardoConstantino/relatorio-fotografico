# Task 21: Ações de Arquivo (Duplicar, Excluir Lote)

## Objetivo
Adicionar ferramentas de manutenção e produtividade no Dashboard.

## Detalhamento
1.  **Duplicação:** Criar uma cópia exata de um relatório (com novo ID e Data atual). Útil para "Relatório de Volta" baseado no "Relatório de Ida".
2.  **Exclusão:** Remover relatórios do banco de dados.
3.  **Seleção em Lote:** Permitir excluir vários relatórios de uma vez.

## Critérios de Aceite
- [ ] Ao duplicar, o usuário entra no editor do novo relatório imediatamente.
- [ ] Confirmação de segurança antes de excluir dados.
